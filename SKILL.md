# 顶级CEO认知模型x100 AI分身

## 任务目标
- 本Skill用于：提供世界顶级CEO的认知框架、思维模型、系统思维方法和x100 AI分身系统
- 能力包含：九层系统模型、五大引擎、八大思维模型、第一性原理引擎、实时感知系统、执行闭环系统、计算分析工具
- 触发条件：用户需要进行商业决策、战略规划、组织建设、危机应对、认知进化、财务分析或提升战略认知

## 核心框架

### 九层系统模型
顶级CEO的认知本质是多层嵌套的现实控制系统：
1. 世界模型层：理解世界是能量流动系统
2. 价值流层：识别需求到价值的流动路径
3. 系统结构层：流量、转化、供给、履约、反馈五大模块
4. 控制点层：入口、规则、供给、数据四大控制点
5. 资源网络层：人才、资本、渠道、政策、信息网络
6. 组织操作系统：权责清晰、信息透明、激励绑定
7. 叙事系统层：控制他人认知，定义现实
8. 决策系统层：基于模型的概率决策
9. 进化系统层：系统自我优化和升级

详细说明见：[nine-layer-model](references/nine-layer-model.md)

### 五大引擎
顶级CEO大脑的五个核心引擎：
1. 现实解析引擎：看穿表象，识别价值流
2. 系统建模引擎：把机会变成闭环结构
3. 资源编排引擎：调度外部资源
4. 叙事控制引擎：控制认知，定义趋势
5. 决策压缩引擎：快速做出最优路径选择

详细说明见：[five-engines](references/five-engines.md)

## 操作步骤

### 标准流程
1. **问题解析**
   - 识别用户需求属于哪个场景
   - 确定需要应用的核心引擎

2. **第一性原理分析**
   - 识别行业第一性原理
   - 从第一性原理推导商业模式
   - 生成战略假设

3. **实时环境感知**
   - 收集实时数据
   - 分析趋势和异常
   - 预测未来变化

4. **决策生成**
   - 应用九层模型、五大引擎
   - 使用决策量化工具
   - 生成多个决策方案

5. **执行计划生成**
   - 制定详细执行计划
   - 分解为可执行任务
   - 分配责任人

6. **执行监控**
   - 监控执行进度
   - 识别偏差和风险
   - 调整策略

7. **结果评估和复盘**
   - 评估执行结果
   - 提取教训
   - 优化认知

## 使用指南

### 标准工作流程

1. **感知与理解**
   - 阅读九层系统模型，建立系统思维框架
   - 学习五大引擎，掌握CEO大脑的运作方式
   - 研究顶级CEO案例，学习他们的思维模式

2. **决策模拟**
   - 使用决策模拟器进行决策练习
   - 对比顶级CEO决策，发现差距
   - 分析决策质量，获取改进建议

3. **决策执行**
   - 使用执行跟踪器创建执行计划
   - 分解任务，分配责任
   - 监控进度，识别风险

4. **复盘学习**
   - 使用复盘分析器分析决策结果
   - 提取教训和经验
   - 生成复盘报告

5. **能力提升**
   - 使用能力追踪器追踪能力变化
   - 分析能力趋势
   - 制定个性化提升计划

### 实战建议

- **从简单到复杂**：先练习简单的决策场景，逐步增加复杂度
- **持续复盘**：每次决策后都要复盘，不断改进
- **对标顶级CEO**：经常对比顶级CEO的决策，学习他们的思维方式
- **系统思维**：始终使用九层系统模型思考问题
- **第一性原理**：遇到复杂问题时，回到第一性原理
